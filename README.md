# Clothes-shop-website
---
## 主要功能
1. 模擬實際服飾網站購買流程，並且可以把想購買的商品加進購物車。
2. 運用瀏覽器cookie來儲存使用者加進購物車的商品，可以新增、刪除。

挑選商品尺寸數量並點擊加入購物車
![image](https://github.com/pt0303ff/Clothes-shop-website/assets/85691121/98145172-11d2-4935-8a62-3c766857d964)

購物車頁面可點擊清空購物車清除所選商品
